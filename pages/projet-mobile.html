Ma Carte Pro (Mobile)

Projet Personnel - React Native

React Native Expo TypeScript AsyncStorage
Voir le code sur GitHub

Le besoin

L'objectif était de créer une application mobile utilitaire permettant de remplacer les cartes de visite papier. L'utilisateur doit pouvoir saisir ses informations une seule fois, les sauvegarder dans le téléphone, et présenter un QR Code scannable pour partager son contact.

Solutions Techniques

  • Persistance des données : Utilisation de AsyncStorage pour sauvegarder le profil localement.
  • Standard vCard : Génération d'une chaîne de caractères formatée pour être reconnue nativement par les contacts iOS et Android.
  • UX Mobile : Gestion du clavier avec Keyboard.dismiss() et défilement fluide.
